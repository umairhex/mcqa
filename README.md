# MCQA - Modern Quiz Application

A modern, accessible, and interactive multiple-choice quiz application with strict state management and real-time feedback.

## 🚀 Quick Start

```bash
pnpm install
pnpm dev
```

## ✨ Features

- **Dynamic Quiz Engine**: Load quizzes via raw JSON input.
- **Strict Flow**: Learn-as-you-go with instant feedback and locked answers.
- **History Tracking**: Auto-saves progress and results.
- **Fully Accessible**: WCAG compliant with keyboard and screen reader support.
- **Dark Mode**: Seamless theme switching.

## 🛠️ Tech Stack

- **Framework**: React 19 + TypeScript 5
- **Build**: Vite 7
- **Styling**: Tailwind CSS 4 + Shadcn UI
- **Icons**: Lucide React
